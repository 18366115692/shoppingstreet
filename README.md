## 划分目录结构
1. 在ssets文件夹下创建css和image文件夹，分别存放css文件和图片资源
2. 在src文件夹下创建一些文件夹
 + views文件夹，存放我们页面的视图
 + common文件夹，存放公共js常量
 + router文件夹，存放路由文件
 + store文件夹，存放vuex数据
 + network文件夹，存放网络请求数据
3. 在components文件夹下创建common和content文件夹
 + common存放所有项目的公共组件
 + content存放当前项目的公共组件
